PROCEDURAL GENERATION IN TERRAIN

Master's Thesis (IN THE MAKING)

OVERVIEW

This Master's thesis project explores the fascinating world of procedural terrain generation, comparing and implementing three distinct algorithms to create realistic and diverse landscapes. The research focuses on analyzing different approaches to procedural generation, from classic Perlin noise implementations to advanced heightmap generation techniques and sophisticated rule-based systems.

The project serves as both a comprehensive academic study and a practical implementation showcase, demonstrating how various procedural generation methods can be applied to create terrain suitable for different applications, from game development to simulation environments. Through rigorous testing and analysis, this work provides valuable insights into the strengths and limitations of each approach.

Each algorithm has been implemented and tested extensively, with detailed documentation of the development process, visual results, and performance characteristics. The following sections present each algorithm individually, showcasing the unique features and capabilities that make them suitable for different types of terrain generation tasks.

IMPLEMENTED ALGORITHMS

01

Perlin Noise Algorithm

The Perlin Noise algorithm forms the foundation of procedural terrain generation, utilizing gradient noise functions to create natural-looking landscapes with smooth transitions and organic patterns. This implementation focuses on multi-octave noise generation, allowing for detailed terrain features at various scales. The algorithm excels at creating rolling hills, mountain ranges, and natural coastlines that feel authentic and visually appealing.

02

Wave Function Collapse

The Wave Function Collapse algorithm, crated in 2016, offers a powerful approach to procedural terrain generation, using constraint-based pattern placement to create diverse and coherent landscapes. Unlike noise-based methods, WFC builds terrain by analyzing example inputs and assembling tiles based on local compatibility rules, ensuring logical and visually consistent terrain features. This implementation focuses on modular tile generation, where the user defines a list of possible tile variants and their adjacency rules. The algorithm then uses these rules to generate terrain that adheres to the specified constraints, resulting in structured and visually appealing environments.
